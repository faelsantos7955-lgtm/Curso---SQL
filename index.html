M1
Fundamentos
O que é SQL, tabelas, linhas, colunas, tipos de dado, anatomia da query.
M2
SELECT em detalhe
Apelidos com AS, escolhendo colunas, comentários.
M3
Filtrando com WHERE
Operadores =, <>, >, <. AND, OR, NOT, IN, BETWEEN, LIKE.
M4
Ordenando e Limitando
ORDER BY ASC/DESC, DISTINCT, LIMIT, OFFSET.
M5
Agregação
COUNT, SUM, AVG, MIN, MAX. GROUP BY, HAVING.
M6
JOINs
INNER, LEFT, múltiplas tabelas, chaves primárias e estrangeiras.
M7
Manipulando Dados
CASE WHEN, funções de string, datas (STRFTIME), COALESCE.
M8
Queries Complexas
Subqueries, CTEs com WITH, Window Functions (ROW_NUMBER, RANK).

Capstone — Case Real
